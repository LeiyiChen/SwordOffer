题目：调整数组顺序使奇数位于偶数前面
>输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

思路：

代码
```python
# -*- coding:utf-8 -*-
class Solution:
    def reOrderArray(self, array):
        # write code here
        l = 0
        h = len(array)
        while l < h:
            if array[l] % 2 == 0:
                array.append(array.pop(l))
                h -= 1
            else:
                l += 1
        return array
```
